# LLM-Generated Bots

Placeholder directory. Populate this manually with bots generated by pasting
the engine README into ChatGPT / Claude / Gemini with prompts such as:
  - "Write a tight-passive 6-max NLHE bot"
  - "Write an aggressive 6-max bot that plays a wide range"
  - "Write a GTO-style bot that estimates pot odds on every street"

## Format

Each bot goes in its own subdirectory:

    harness/opponents/llm_generated/<bot_id>/bot.py

`bot.py` must expose a `decide(game_state: dict) -> dict` function.

## Including in Sweeps

LLM-generated bots are NOT automatically included in the training pool.
To include them, either:
  1. Add paths to `harness/opponents/registry.py` under `_LLM_TRAIN`
  2. Pass a custom `--opponents pool.json` to `harness/cli.py compare`

## Heldout Protocol

Pick 2-3 of these bots as held-out validation opponents (never used during
the Optuna sweep, only for final evaluation). Add them to
`harness/opponents/heldout/` once you decide which ones to hold out.
